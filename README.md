# 💰 Vibe Expense Tracker (記帳小幫手)

這是一個使用 Python 與 Streamlit 構建的現代化記帳應用程式。專注於區分「現金」與「信用卡」的支出管理。

## ✨ 功能特色

- **雙軌記錄**：清晰區分現金 (Cash) 與信用卡 (Credit Card) 消費。
- **即時儀表板**：首頁直觀顯示各類別總支出金額。
- **彈性過濾**：
  - 支援按「月份」查看消費紀錄。
  - 支援切換只顯示「現金」或「信用卡」明細。
- **資料持久化**：自動將紀錄保存為 CSV 檔案，簡單且易於備份。
